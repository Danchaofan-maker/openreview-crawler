02 · 维度相关性与真实自由度

Pearson相关系数热图（全量30k）

发现：10个维度实际只有 3 条真实独立轴
相关性分析（Pearson + Spearman 一致）揭示了严重的多重共线性。全量30k数据与Codex的600篇样本结论一致：

· mr × tn × md 两两 r = 0.82–0.90，构成同一"理论深度"轴
· er 是这个轴的镜像（r ≈ −0.83 至 −0.77）
· ei / sg 与理论轴正相关（r ≈ 0.53–0.66），信息冗余
· ar 与所有维度相关性接近0，是唯一真正独立的维度
· cc 中等独立（r ≈ −0.39 至 −0.34）
· cs 几乎完全独立（r < 0.13 与所有维度）
轴 A · 理论强度轴（主轴）
mr 正向 tn 正向 md 正向 er 反向 ei 附带 sg 附带
解释方差最多。知道一篇论文的mr，基本可以预测tn、md、er的值（r²≈0.65–0.81）。这六个维度本质上是同一个"这篇论文有多理论化"的反复测量。
轴 B · 假设现实度轴（独立）
ar 独立
与理论强度轴正交。顶级纯数学论文未必假设更现实——"数学严谨"和"假设接地气"是不同的东西。这是唯一真正提供独立信息的维度。
轴 C · 算力规模轴（弱独立）
cc 弱独立
与理论轴负相关（r≈−0.39），但分布单峰、区分力弱（std=1.40，p75−p25=1.0）。对精读筛选价值有限，但在识别"重算力大规模实验"类论文时有意义。

04 · UMAP 空间验证：9维 → 2维的几何结构

分离比 1.44 — 特征空间有真实几何结构
8000篇论文（分层抽样，全部4类）降维后：

intact 质心 [10.14, 8.22]，扩散半径 0.98
partial 质心 [6.17, 8.23]，扩散半径 1.77
质心距离 / 扩散之和 = 3.96 / 2.75 = 1.44

大于1说明两类论文在特征空间里有清晰的分布分离，不是随机噪声。当前 rules 的线性切割有几何依据。

UMAP 分离度可视化（示意）

重要限制：Euclidean 距离在序数格点上是近似
分数是半整数离散格点（锚定率100%）。用欧式距离计算"2.5分和3.0分的距离"与"2.0分和2.5分的距离"相等这个假设是近似的，因为模型对格点的使用频率不均匀。

然而，Spearman相关（序数）与Pearson（等距）结论高度一致（差异 < 0.02），说明在30k这个规模下，等距假设引入的误差被样本量稀释了。UMAP和当前rules继续用欧式距离是合理的，但应避免对单格点差异（如3.5 vs 4.0）做精确性解读。
absent 论文不是废纸
97.1% 的 absent 论文有非零分数，er（96.6%）、cc（95.8%）、sg（90.9%）覆盖率极高。absent 在特征空间里是"低mr + 高er"的角落，和 broken 有重叠但不完全相同。

之前 UMAP 丢掉 absent 是错误的——现在已修正，把 absent 纳入投影（灰色点）。

05 · 打分可信度：重复实验结果（9篇 × 5次）

各维度组内标准差（同一篇论文5次打分的波动）

可信轴（std < 0.75）
mr（0.515）和 cs（0.623）。mr 是整个体系里最稳定的维度，适合作为主要筛选轴的基础。
中等轴（0.75 ≤ std < 1.0）
md（0.701）、cc（0.708）、er（0.813）、tn（0.803）。可用于筛选但不应对单格点差异过度解读。
噪声轴（std ≥ 1.0）
sg（0.985）、ei（1.254）、tea（1.276）、ar（1.269）。高方差意味着同一篇论文多次打分差距可达2分。不适合精确阈值切割，只适合大范围过滤或辅助权重。

注意：ar 既是噪声轴（重复实验std=1.27）又是独立轴（与其他维度不相关）——独立不代表可靠。
ig 分类一致率：0.778
5次打分里 intact vs partial 边界有约22%的随机翻转，集中在mr=4~7的中间地带。force_keep_hr 的逻辑正是为此设计的。

06 · 如何建立特征空间：三条路径对比

路径 A · 原始10维欧式空间

直接用10个维度做距离计算（UMAP已在用）。

优点：无需转换，计算简单，UMAP分离比1.44已验证有效。

缺点：多重共线性导致"理论强度"被隐式加权5~6倍（mr+tn+md+er_inverse+ei+sg全都携带同一信息），ar和cc被稀释。不是"均匀"的特征空间。
现状：UMAP已用此方案，结果可接受。

路径 B · PCA降维后的真实轴

先做主成分分析，取前3~4个主成分（估计解释方差 ~85%），再在PC空间做距离计算。

优点：消除共线性，每个轴携带独立信息，距离度量更"均匀"。PC1≈理论强度，PC2≈假设现实，PC3≈算力/规模。

缺点：主成分的可解释性下降，不直观；茶叶和ar的噪声会被PCA部分吸收。
推荐用于：聚类分析、论文相似度计算。

路径 C · 手工双轴（mr轴 × ar轴）

直接用 理论强度综合分（mr的加权组合）作为X轴，ar 作为Y轴，构建2D人类可读空间。

优点：完全可解释，X=数学骨架强度，Y=假设现实度。四象限直接对应四类论文价值判断。与现有rules体系兼容。

缺点：丢弃了er/ei/sg/cc的独立信息（虽然不多），ar的高噪声（std=1.27）污染Y轴。
推荐用于：人类浏览、向他人解释、制图。
对现有 rules 体系的含义
Rules 用的是 集合论 + 不等式（超平面切割），隐含了欧式/线性空间假设。这在数学上是正确的——每条规则是高维空间里的一个半空间，多条规则的交/并是多面体。只要我们想要的论文确实落在某个区域，规则就是有效的。

多重共线性不影响rules的逻辑正确性，只影响"效率"：R2（mr < 3）和R5（ig=partial AND mr < 8.5）在维度相关的情况下有部分重叠，但不会产生错误排除。可以继续用当前rules体系，但在写新规则时应优先用 mr 而不是同时用 mr + tn + md（三者冗余）。

07 · 核心结论与后续行动

1
10个维度 → 3条真实轴。理论强度（mr主导）、假设现实度（ar独立）、算力规模（cc弱）。现有rules的实际信息冗余度较高，但逻辑上没有错误。
2
特征空间有真实几何结构。UMAP分离比1.44证明intact和partial在9维空间里是可区分的，不是分布完全重叠的随机类别。rules在这个空间里的线性切割是有效的。
3
absent不应该被一刀切丢弃。97%有非零分数，er/cc/sg维度有真实多样性。R1（ig=absent直接淘汰）对于我们的目标（精读数学骨架）是合理的，但absent论文在其他维度上存在独立的结构，未来可单独分析。
4
写新rules时的优先级：① mr（最可靠 + 最有区分力 + 双峰谷值~4.0提供自然切割点）② er（mr的镜像，er≤4与mr≥6高度等价）③ ig（最清晰的类别标签）④ tn/md（辅助，与mr冗余）⑤ ar/cc（独立信息但噪声大）⑥ 避免 cs/ei/sg作为主要切割轴。
