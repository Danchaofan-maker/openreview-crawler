# 字段速查

## 打分维度（0–10）

| 缩写 | 中文       | 一句话                         |
|------|------------|-------------------------------|
| mr   | 数学严谨   | 推导链有多严格，有无漏洞       |
| md   | 数学深度   | 用到的数学工具有多深           |
| tn   | 理论新颖   | 理论贡献本身有多新             |
| ar   | 假设现实   | 假设离现实有多近（高=接地气）  |
| er   | 经验依赖   | 结论多大程度靠跑实验支撑       |
| tea  | 理论实验咬合 | 理论预测和实验结果对得上吗    |
| cc   | 算力门槛   | 复现这篇需要多少算力           |
| ei   | 认识论意图 | 这篇想证明什么 vs 想展示什么   |
| sg   | 适用范围   | 结论能推广到多广的场景         |
| cs   | 置信度     | 模型自评：我对以上打分多有把握 |

## 分类标签

| 缩写  | 中文           | 取值                                    |
|-------|----------------|-----------------------------------------|
| ig    | 逻辑链完整性   | intact / partial / broken / absent      |
| mk_f  | 营销包装       | True = 有明显炒作/夸大措辞              |
| hr_f  | 人工复查       | True = 边界模糊，建议人看               |
| fuse  | 骨架缺失       | True = absent，分数不可信               |

## ig 含义

- **intact**  — 推导链完整，结论可从前提严格导出
- **partial** — 有证明但有跳步 / 部分结论靠实验补
- **broken**  — 声称有理论但推导明显不成立
- **absent**  — 纯实验 / 工程，没有数学骨架
