## 当前版本: v0.8

### 重大决策记录

- **关闭 thinking 模式**：实测 thinking 占 output token 85%，关闭后单篇成本从 ¥0.024 降至 ¥0.003。质量影响在小样本上可接受。
- **字段全部缩写**：mr/tn/md/... 加 s/r/f/ig/lc/dm/mk/hr/osn/rr。下游兼容旧 v0.7 长字段名通过 helper 函数实现。
- **双 schema 设计**：紧凑（fuse=true，扁平）+ 完整（fuse=false，嵌套）。`fuse`/`rr` 必须放在 JSON **开头**，否则 non-thinking 模型无法决定后续 schema 走嵌套还是扁平。
- **不传 venue/year**：避免会场声誉偏见污染评分。下游若需 venue 维度的统计分析，从原始 metadata 关联即可。
- **规则取消文字标签**：原本叫"刷榜的来了"等带感情色彩名字，模型会因名字暗示偏向某一规则。改为纯数字编号（rule 1–6）+ 数学条件。
